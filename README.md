# ScriptFunnel - Premium Script Marketplace

A high-fidelity single-page website for showcasing and selling Google Apps Scripts and business automation solutions.

## ✨ Features
- **Premium UI**: Modern, responsive design with glassmorphism and smooth animations.
- **Dynamic Content**: Scripts are rendered dynamically from `script.js`.
- **Search & Filter**: Real-time searching and category-based filtering.
- **Ready for Render**: Optimized for static site deployment on Render.com.

## 🚀 Deployment
This site is designed to be deployed as a **Static Site** on Render.

1. Connect this repository to [Render](https://render.com).
2. Render will automatically detect `index.html` and deploy.

## 🛠 Tech Stack
- HTML5
- Vanilla CSS
- Vanilla JavaScript
- Font Awesome (Icons)
- Google Fonts (Inter)
